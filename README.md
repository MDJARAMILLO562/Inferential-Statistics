# Hypothesis Testing Project
Academic project focused on inferential statistics and hypothesis testing.

## Objective
To apply statistical hypothesis testing methods to analyze and compare groups based on quantitative data.

## Methods
- Exploratory analysis
- Assumption checking
- Hypothesis testing (parametric and/or non-parametric tests)
- Interpretation of results

## Tools
- Python
- Pandas
- SciPy / Statsmodels
- Matplotlib / Seaborn
